# Karakter Tabanlı Hayatta Kalma Simülasyonu

Bu proje, C programlama dili kullanılarak geliştirilmiş metin tabanlı bir hayatta kalma simülasyonudur. 
Oyuncu, sağlık, enerji, yemek ve sığınak gibi durum değişkenlerini yöneterek hayatta kalmaya çalışır.

## Kullanılan Komutlar
- A: Avlan
- S: Sığınak Ara
- R: Dinlen
- E: Durumu Göster
- F: Tehlike Dalgası (FOR döngüsü)
- P: Geçiş Mekanizması (DO-WHILE döngüsü)
- X: Oyundan Çık
